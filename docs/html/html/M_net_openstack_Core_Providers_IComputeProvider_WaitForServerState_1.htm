IComputeProvider WaitForServerState Method (String,  String ,  String , Int32, Int32, Action Int32 , String, CloudIdentity) openstack.net API Reference Documentation

[This is preliminary documentation and is subject to change.]

Waits for the server to enter a set of specified states.
Remarks
NOTE: This is a blocking operation and will new return until the server enters either the expected state, an error state, or the retry count is exceeded
